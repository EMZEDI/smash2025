# SMASH

### Stack
- Hugo
- Tailwind
- Netlify

### Local install
1. `brew install hugo`
2. `hugo server --disableFastRender`

### Deploy
Auto deployments are set up pointing to the main branch. Once your work is ready, merge it in and wait.
